Fit.toSingleLine( element, width, minimumSize )

For multiline text, this will try it's best to fit on one line, scaling all the way down to the minimum size ( defaults to 6px )

Fit.longestWord( element, width, minimumSize )

Do not let long words fall outside of thin elements

Fit.inHeight( element, height, minimumSize )

For fixed heights, do not let the text vertically overflow

Fit.countLines( element )

This will attempt to work out how many rows this text vertically spans.

Instructions

First embed JS path code in html

Then using a HTMLElement, call on of the above methods

// Fetch an element from the DOM
var element = document.getElementById('headline');

// Fit the text in an element to fit on one line
Fit.toSingleLine(element);

// Max Width is 400px
Fit.toSingleLine(element,400);

// Max Width is 400px, minimum font size is 12px
Fit.toSingleLine(element,400,12);
